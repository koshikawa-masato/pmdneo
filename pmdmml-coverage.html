01. 概要

このページは PMDMML 142 command base table を実装進捗の基準にする。基準表は neo-sisters Issue #109 にあり、詳細な coverage markdown は neo-sisters 側の knowledge document として管理されている。

PMDNEO には parallel lineage が 2 系統ある。main は nullsound-based driver を起点に PMD / PMDAES 系の command support を拡張してきた lineage、develop は standalone scratch driver として共通 dispatch、hook table、14 part 並行 dispatch を組み直している lineage である。
