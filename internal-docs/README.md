# Internal documents

このディレクトリは、PMDNEO の内部作業資料を置く場所です。

ここに置くもの:

- AI 継続用の高密度メモ
- ChatGPT / Codex との壁打ち記録
- 長い trace や verify script の詳細出力
- commit ごとの作業ログ
- 公開 `docs/` に移す前の下書き

ここに置かないもの:

- 人が最初に読むべき設計資料
- 現在有効な仕様
- README から参照される正式な手順

`internal-docs/` の中身は原則として git 管理外です。この `README.md` だけを置き場の説明として管理します。

内部資料から公開資料へ移すときは、内容をそのまま貼り付けず、短く平易な文章に要約してください。
